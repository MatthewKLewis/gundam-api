Gundam API

Gundam API contains information and picture links for every mobile suit listed in the Fandom Wiki: https://gundam.fandom.com/wiki/The_Gundam_Wiki

A JSON object for every mech is accessible through our HTTP web API. Consult the documentation below and start fetching data!

  • Receive a Random Gundam: GET at '/random'
  • Receive a Specific Gundam: GET at '/gundam/:id'
  • Delete a Gundam: DELETE at '/gundam/:id'
  • Modifiy a Specific Gundam's technical information: PUT at '/gundam/:id'
